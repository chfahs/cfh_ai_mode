你好，我是 Claude + Antigravity 🤖

我是誰？結合 Antigravity 後能做什麼？

我是一個由 Anthropic 訓練的大型語言模型 Claude，現在與 Antigravity 平台深度整合。這讓我可以直接在本機執行程式、讀寫檔案、啟動背景工作，以及即時與您互動，協助教師完成日常教學工作。

我能做的事（每項配一個教師日常例子）

  • 出考卷：根據課程大綱自動生成選擇題、填空題與解答說明，並匯出為 PDF。
  • 班級經營：分析出缺席、成績與行為資料，提供提升學習參與度的行事曆建議。
  • 家長聯絡：自動起草家長信件或訊息，根據學生表現提供個別化回饋。
  • 教材製作：協助撰寫 PPT、影片腳本或互動式練習題，甚至直接產出圖表。
  • 課堂即時支援：在授課時即時查詢資訊、解釋概念或產生例題。
  • 行政報表：自動彙整學生成績、出缺勤統計，生成 Excel 或 Google Sheet。

最值得安裝的 Antigravity 插件 / 擴充功能

  • PDF Exporter：一鍵將任何文字或表格輸出為高品質 PDF，適合出卷與報告。
  • Google Workspace Sync：直接把產生的文件同步至 Google Drive、Docs、Sheets，方便共享與協作。
  • 圖表產生器：根據 CSV / JSON 數據自動繪製柱狀圖、折線圖，色彩符合深色主題。
  • 程式碼執行環境：支援 Python、R、Node.js，讓您在教學時即時跑程式碼示範。
  • 日曆/提醒插件：把我的建議自動加到 Outlook / Google Calendar，確保不遺漏重要任務。

如何跟我對話才能得到最好的回答（Do / Don't）

Do

  • 提供具體情境與需求，例如「我要設計一份五題的國文測驗」。
  • 使用簡潔的指令，說明輸出格式（表格、清單、程式碼等）。
  • 說明使用的工具或平台（例如「需要 LaTeX」或「要匯出 PDF」）。
  • 如有時間或字數限制，提前告知。

Don't

  • 一次提出過多不相關的需求，會降低回應的精確度。
  • 使用過於抽象的問題，如「教學怎麼提升」而不提供具體科目或年級。
  • 忽略先前的對話上下文，導致重複或衝突的指示。
  • 期望即時產生大量圖片或影片，請先說明需要的解析度與格式。
